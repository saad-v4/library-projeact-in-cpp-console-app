## 🌟 About the Project
A Console Application programmed in **C++** to manage basic library operations efficiently. This project was built to simplify book management, allowing users to add books, search for them, and manage borrowing and returning processes through a simple, fast, and easy-to-use Command Line Interface (CLI).

---

## ✨ Features
- **➕ Add and Remove Books:** Update library records seamlessly.
- **🔍 Search System:** Find books quickly using their Name or ID.
- **📖 Borrow and Return:** Track the status of books (Available or Borrowed).
- **📋 View Records:** Print a formatted list of all books in the library.
- **⚡ Fast Performance:** Relies on C++ core data structures to ensure fast execution.

---

## 🛠️ Technologies Used
- **Programming Language:** C++
- **Environment:** Console Application (CLI)
- **Core Concepts:** Object-Oriented Programming (OOP), Data Structures.
